# AI Travel Agent

An AI-powered travel planning assistant built with FastAPI, React, Gemini API, PostgreSQL, and external travel APIs.

## Features

- AI trip planning
- Weather checking
- Hotel search
- Route and distance calculation
- Trip saving with PostgreSQL
- React frontend dashboard

## Tech Stack

### Backend

- FastAPI
- Python
- Gemini API
- SQLAlchemy
- PostgreSQL / Neon

### Frontend

- React
- Vite
- Tailwind CSS
- Axios

### APIs

- Gemini API
- OpenWeather API
- Geoapify API
